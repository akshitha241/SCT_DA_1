# SCT_DA_1
Excel Sales Dashboard created using Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting as part of SkillCraft Technology Data Analytics Internship Task 1.
## Features
- Total Sales Analysis
- Profit by Category
- Sales by Region
- Sales by Category
- Profit by Sub-Category
- Interactive Slicers
- Conditional Formatting

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

- Excel Dashboard Workbook (.xlsx)
